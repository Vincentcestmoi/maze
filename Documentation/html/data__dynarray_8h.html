Description détaillée

Implémentation d'un tableau dynamique de nombres entiers.

La structure de données est identique à celle utilisée pour implémenter une pile.

Remarques
Cette structure de données est utilisée régulièrement dans le projet. En particulier, il est possible de s'en servir lorsque l'on a besoin d'une pile.
