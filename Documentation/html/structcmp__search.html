Description détaillée

Implémentation d'un chemin complexe: il s'agit d'une séquence de chemins simples (sans cycle) qu'on stocke dans un tableau.

Attention
Bien que ce ne soit pas imposé par le type, deux chemins simples consécutifs sont censés se suivre. C'est-à-dire que le point d'arrivée d'un chemin doit être égal au point de départ du chemin qui le suit dans la séquence.
