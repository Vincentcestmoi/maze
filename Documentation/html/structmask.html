Description détaillée

Le type utilisé pour représenter un masque.

Remarques
Les cellules sont indexées par les entiers allant de 0 à hsize*vsize-1. Elles sont ordonnées ligne par ligne du Nord au Sud. La cellule d'indice 0 se trouve au coin Nord-Ouest du labyrinthe. Les cellules allant de 0 à hsize-1 constituent la première ligne au Nord du labyrinthe, et ainsi de suite.
Attention
On supposera toujours qu'un masque contient au moins une cellule non-masquée (cette propriété doit être vérifiée à la lecture d'un fichier masque).
